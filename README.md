## Kaguya-Bot

## Pre-required
1. Node.js [install here](https://nodejs.org/en/download)
2. Git [install here](https://git-scm.com/downloads)

## CLone this project
```sh
git clone https://github.com/FaizBastomi/kaguya-bot.git
```

## Install Depencies
```sh
npm install
```

## Run Bot
```sh
node index.js
```
